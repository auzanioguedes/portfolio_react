## 📚 Seções

O site é composto por três seções:

- **Home:** Apresentação com nome e formação acadêmica;
- **Sobre mim:** Descrição sobre quem sou e minha missão;
- **Projetos:** Projetos renderizados dinamicamente via componente React.

---

## 💼 Tecnologias utilizadas

- React 18
- Vite 6
- HTML5
- CSS3
- JavaScript (ES Modules)

---

## ⚙️ Funcionalidades

- Menu responsivo com hamburger para mobile
- Projetos renderizados dinamicamente via componente React
- Layout responsivo com Flexbox e CSS Grid
- Rodapé com link para WhatsApp

---

## 🚀 Como executar localmente

```bash
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev

# Gerar build de produção
npm run build

# Pré-visualizar o build
npm run preview
```
